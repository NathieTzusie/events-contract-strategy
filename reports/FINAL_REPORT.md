### 核心洞察

## 2. 最终策略：三级规则体系

### 规则来源
从 GradientBoosting 特征重要性分析中提取：
- **G1_HigherHighs** (40% 重要性): 摆动点衰竭检测
- **F2_ZScore** (6%): 收益率的统计极端值
- **C4_TF_ADX** (4%): 高时间级 ADX 趋势确认
- **H2_SMA20_100** (4%): 长期趋势方向

### PRIMARY (高置信度, 85%)
```
LONG:  HigherHighs < 0  AND  ZScore < 0  AND  ADX_TF > 0
       (摆动低点衰竭)      (近期弱势)       (高TF看涨趋势)

SHORT: HigherHighs > 0  AND  ZScore > 0  AND  ADX_TF < 0
       (摆动高点衰竭)      (近期强势)       (高TF看跌趋势)
```
→ 高时间级趋势确认的反向交易。**最可靠。**

### SECONDARY (中置信度, 75%)
```
LONG:  SMA20_100 > 0  AND  HigherHighs < 0  AND  ZScore < 0
       (长期趋势看涨)     (摆动低点衰竭)       (近期弱势)

SHORT: SMA20_100 < 0  AND  HigherHighs > 0  AND  ZScore > 0
```
→ 长期趋势中的回调买入/反弹卖出。

### TERTIARY (基础置信度, 65%, 仅30m)
```
LONG:  HigherHighs < 0  AND  ZScore < 0
SHORT: HigherHighs > 0  AND  ZScore > 0
```
→ 纯反转信号，需要波动率低时使用。

## 7. 关键限制与风险

1. **回测周期仅 16 个月** — 未覆盖完整牛熊周期
2. **BTC only** — 需验证 ETH 等其他品种
3. **事件合约流动性** — 取决于交易所
4. **滑点/价差** — 未计入，但事件合约按到期结算价，影响较小
5. **极端行情** — 闪崩/暴拉可能导致规则短暂失效
6. **10m 数据为重采样** — 非原始 10m 数据
